# HRMS Frontend

Next.js frontend for the HRMS application.

## Setup

1.  **Environment Variables**:
    Ensure `.env.local` exists.

2.  **Install Dependencies**:
    ```bash
    pnpm install
    # or
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    pnpm dev
    # or
    npm run dev
    ```
    App runs at: `http://localhost:3000`

## Build

```bash
pnpm build
pnpm start
```
